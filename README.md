#  Global Job Market Analysis (Data Analyst Project)

##  Project Overview

This project analyzes the global data job market using real-world job posting data to uncover hiring trends, salary insights, skill demand, and remote work patterns.


---

##  Objectives

* Identify most in-demand data roles globally
* Analyze salary trends by role, country, and skills
* Understand remote vs onsite job distribution
* Discover top technical skills driving hiring demand
* Build an interactive Power BI dashboard for insights

---

##  Tech Stack

* Python (Pandas, NumPy, Data Cleaning)
* SQL (MySQL – Data Analysis)
* Power BI (Dashboard & Visualization)
* Excel (Initial exploration)

---


##  Key Insights

* Data Engineers and Data Analysts dominate hiring demand globally
* SQL and Python are the most in-demand technical skills
* Remote jobs contribute significantly to global hiring trends
* Senior roles command the highest salaries
* Country demand impacts compensation levels
* Cloud & ML skills lead to higher salary bands
